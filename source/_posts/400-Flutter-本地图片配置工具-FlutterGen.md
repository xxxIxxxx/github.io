# [github -> FlutterGen](https://github.com/FlutterGen)
# 安装
```
# 推荐使用 Homebrew 安装
$ brew install FlutterGen/tap/fluttergen
```

#使用前先把 assets: 配置好
```
assets:
    - assets/images/cat.png
```
# 使用
```
$ fluttergen -c example/pubspec.yaml
```
